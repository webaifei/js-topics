## generator总结

> 协程在js中的实现

特性：
1. 可以暂停函数执行,交出控制权
2. 执行函数返回一个有限状态机

注意点：
1. yield 默认返回值总是undefined，定义不同的状态
2. next 方法移动内部指针到一个状态，返回一个对象，包含value属性（yield后面的表达式的值），和一个是否遍历完成done属性
3. next可以接收一个参数，这个参数会被当做上一个yield后面表达式的返回值。
